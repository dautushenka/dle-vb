DLE + vBulletin v2.2.0
Установка » vBulletin
1. Открываем файл /includes/config_dle.php и заполняем туда данные БД сервера ДЛЕ (это файл конфига БД сайта за исключением строки инициализации объекта БД).
2. Заходим в админку форума Продукты и модули->Управление продуктами
Выбираем "Добавить/импортировать продукт"
Выбираем и загружаем файл product-dle_integration_utf8.xml или product-dle_integration_cp1251.xml в зависимости от вашей кодировки форума
3. Заходим в настройки интеграции на форуме
Настройки->Основные настройки->Интеграция с DataLife Engine
